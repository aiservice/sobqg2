青城山，三清观。

观外，一个身着蓑衣斗笠的老者轻轻叩了叩门。

此刻正下着瓢泼大雨，他似是要找一个避雨的地方。

过了半晌方是有一个小道士打开了门。

“这位善信，今日道观闭观了，你要上香还是去别处吧。”

那身着蓑衣斗笠的老者摇了摇头道：“我是来求避一避雨的。这雨下的太大了，一时半会也不见放晴的样子。下雨天赶山路太危险了，还请行个方便。”

那小道士一听直是愣住了。

对方说的似乎也很有道理。

这下着瓢泼大雨的，他要是不让对方进来确实也不合情理。

“如此，请先进来吧。我去与道长说一声。不过待雨停了你就得走，我们还得炼丹呢。”

“自当如此。”

那身着蓑衣斗笠的老者点了点头。

进入三清观后，小道士走在前面引领，他把那身着蓑衣斗笠的老者引到了一处厢房后遂沉声道：“我得去跟道长先说一声，您且在这里等着。”

“等等。”

那身着蓑衣斗笠的老者掏出一块玉玦，放到了小道士的手中。

“请把这个给你们道长看看。”

“嗯？”

小道士听得一愣，不过他还是点了点头，拔步而去。

身着蓑衣斗笠的老者遂径直坐下，闭目养神。

约是一盏茶的工夫后，小道士去而复返，很是恭敬的冲那蓑衣斗笠的老者作揖拱手。

“我家道长有请，还请您老跟我走一趟。”

身着蓑衣斗笠的老者微微颔首道：“好。”

...

...

“师叔，果然是您！我就说这块玉玦除了您不会有其他人拥有！”

三清观的道长平阳子见到这身着蓑衣斗笠的老者之后立刻激动的迎上前去。

“师叔，您不是在长安修行呢吗，怎么会来青城山？”

原来，这身着蓑衣斗笠的老者不是旁人，正是青莲道长吴全义。

他将斗笠摘下，笑了笑道：“怎么不欢迎老夫来看看？再怎么说，老夫也是从这青城山走出去的，回来看看不是很正常吗？”

“没有，没有。我不是这个意思。我是说，师叔怎么会有工夫来青城山游历的？”

“瞧你这话说的。老夫的道观在长安城不假，可是老夫又不是没有腿脚。长安城待的无趣，老夫便出来看看。想来这青城山，自然也就来了。哪里有这么多所谓。你小子，最近却是变得圆滑的紧。不像是老夫刚走那一会，三棍子打不出一个闷屁。”

“额。师叔，我现在好歹也是道长了。您多少给我留点面子啊。”

平阳子被青莲道长吴全义说的直是面色潮红，一时间憋的尴尬极了。

“好了好了，老夫也不跟你在这里打趣了。实话跟你说，老夫觉得长安城没有什么意思，又起了思乡之情，便想着回蜀地来养老。这三清观可还有老夫的位置？”

“当然，当然...快，快去给师叔收拾一间上好的厢房来。”

平阳子闻言连忙点头，随即冲身旁的小道士吩咐道：“一切吃穿用度皆按我的标准来。”

...

...

青城山三清观旁有一座莲花湖。

莲花湖并不算大，但是里面有不少的游鱼。

青莲道长吴全义闲来无事便开始坐在湖畔钓鱼。

平阳子小心翼翼的取了一个竹篓在一旁侍奉。

“师叔，你这次回来就不走了吧？”

“应该是。老夫在长安待的时间太久了，觉得哪里都没有意思。索性回到这青城山养老。不得不说，这么多年过去了，青城山还是没有什么变化。三清观还是这个样子，这莲花湖也是当时的模样。小地方就是好啊，若是在长安城，别说十几年的时间里。便是十几天过去，怕是都会变个样！”

“额，可是长安城是人人都向往的地方啊。这天底下有数不清楚的人挤破了头都想要进到长安城呢。有一句话怎么说来着，学得文武艺，卖与帝王家。”

平阳子顺着青莲道长的话头说道：“像您老这样的人物，要是留在长安城，怎么也得是国师级别的人物。就这么回来了，确实是太可惜了。”

“有甚么可惜的。老夫又不想给皇帝老儿卖命。之前在长安城的时候也是大部分时间都住在终南山。什么学得文武艺，卖与帝王家。这都是读书人编出来的。咱们道门中人是出家人，功利心不能这么重。你小子还是得多修修心。刚刚你说的那番话若是传将出去怕是要让人笑掉大牙呢。”

青莲道长吴全义翻了翻白眼道：“这样吧，你也取来一个鱼竿来陪老夫钓鱼好了。”

...

...

平阳子很是无奈。

青莲道长是他的师叔，辈分在他之上。

青莲道长的话他当然得听。

“好吧，师叔请稍候。”

说罢他便折返回三清观。

一盏茶的工夫后，平阳子也取来了一副鱼竿和一只竹篓，坐到了青莲道长的身边。

“师叔，您是什么时候喜欢上钓鱼的？我记得当初在青城山的时候您可是没有类似的爱好的。”

“瞧你这话说的，人的习惯难道还不能改吗？老夫离开青城山的时候不过是四十出头。那个时候还是踌躇满志，想要大展拳脚。那种时候老夫的心态怎么会像现在这样闲适？又怎么会喜欢钓鱼？可是现在不同了，现在老夫都快六十了。六十而耳顺知道不？你若是到了老夫这个年纪，也该是这副心境了。不过无所谓，你现在的年纪还不可能明白这些的。”

青莲道长吴全义显然丝毫不在意平阳子刚刚的那番话。

“就说这钓鱼吧。在不同的地方钓也完全不同。老夫现在的心态就是绝对的平和。一般人完全无法理解。换句话说，老夫钓鱼是愿者上钩。钓不钓的上鱼不关键，关键点是在于钓鱼的过程。只要老夫能够享受到这个过程就是非常完美的了。”

青莲道长吴全义说罢翘着二郎腿，很是得意的问道：“话又说回来了，你小子这么多年修为境界可有增长啊？”

“额，不瞒您说，我这些年来，已经突破至二品境界了，可距离一品却总有一段距离。”

...

...

“二品？你现在才二品修为？”

青莲道长吴全义听到这里之后不禁翻了下白眼。

“老夫可是记得当初离开青城山的时候你就已经是四品了，合着这么多年过去了，你的修为境界才升了二级？”

“额，说来惭愧。我感觉自己距离一品那扇门越来越远了，一开始的时候还能够触碰到那扇门的门槛，现在竟然连门槛都摸不到了。”

“罢了。也算是你小子运气好，赶上了老夫回来。既然老夫回来了，自然不能眼睁睁的看着你在一品的大门外打转。这段时间你便跟着老夫好好的修行。应该用不了多久就能够破境了。”

“啊那真的是太感谢师叔了！”

平阳子闻言直是一阵狂喜。

“你先别急着谢老夫，待破境至一品后再谢也不迟。再说老夫这么做也不光是为了你，更是为了三清观。正所谓矬子里面拔将军。若是老夫不把你推到一品修为，还能指望谁？你们这些后辈们啊真的是一个个的不中用。”

青莲道长吴全义却是越想越气。

“师叔教训的是。确实我们太不争气了。不过有师叔的点拨，相信用不了多久我们就能够突破自己的极限，百尺竿头更进一步了。”

平阳子现在自然是顺着青莲道长的意思说。

毕竟一品和二品乃是天壤之别。

若是他有幸达到一品境界，就可以在道门之中占据一个更加有利的位置。

不管从什么角度看，他都是稳赚不赔的。

...

...

青莲道长吴全义笑了笑道：“你小子还算是识趣。不过老夫要告诉你一点，千万不要急于求成。修行要的是一个水到渠成。如果你过于焦急的话就可能会适得其反。虽说道门修行对于气息的控制相当到位，可还是会存在一些变数的，并非完全没有走火入魔的可能。”

“受教了。”

平阳子点了点头道：“对了师叔，您现在已经是什么境界了？”

“老夫已经晋升超品了。”

青莲道长吴全义背负双手很是得意的说道。

他一直在等平阳子问这句话。

这小子还真的是很上道。

“啊，师叔您已经达到超品修为了？如今这世间能够达到超品境界的不会超过十人吧？”

平阳子显然是被震惊到了。

“差不多吧，两只手数的过来。”

青莲道长吴全义清了清嗓子道：“不过超品境界亦有差距。有的超品修行者是拥有飞升的机会的，只是他们自己强行压着境界不去飞升。”

“但是有的超品则是刚刚踏过那个门槛，距离飞升还有很远的距离。”

“那师叔您是...”

“老夫差不多是中间的部分吧，不算是最强的，但也不是最差的。”

青莲道长吴全义清了清嗓子道：“你小子不要好高骛远，先把一品修为突破了再说。武道修行没有捷径可走，一定要一步一个脚印才能够达到更强大的境界。”

青莲道长吴全义这次回到青城山也是有自己的私心的，他是希望道门弟子能够在修为上有所精进，至少不要被儒家弟子甩的太远。

...

...

青莲道长吴全义还是很重视道门的秩序的。

不管是任何人，都应该尽可能的去按照一定的秩序修行。

所以他将准备好的道门宝典送到了平阳子的手中。

“你先按照这个典籍来修行吧，能够达到什么样的程度就先练到什么程度。需要老夫出手的时候老夫会来指点你的。”

平阳子听罢喜不自胜。

“多谢师叔。您放心，我一定会潜心修行的。”

平阳子自从成为三清观的道长后他就一直在挖掘自己的潜能。

因为各种各样的因素他无法捅破那层窗户纸。

如今遇到了青莲道长的指点，绝对是他破境的最佳时机。

所以平阳子绝对不会错过这个绝佳的机会的。

“嗯，那你就先练着好了。老夫要开始钓鱼了。没有旁的事情，不要来打扰老夫。除非是你遇到了修行上面的事情，老夫可以给你指点一二。”

说罢，青莲道长吴全义又开始钓鱼了。

对他来说，这绝对可以算是最惬意的时刻。

...

...

“没有想到过了这么久，这莲花湖的鱼还是这么精明。老夫都已经用了这么多的饵料，它们还是不上钩。”

青莲道长吴全义越想越气，直接将鱼竿丢到了一旁。

“罢了，罢了。老夫要先去吃个火锅耍耍。”

既然回到了蜀地，火锅是一定要吃的。

在青莲道长吴全义看来，比起长安城，这里的火锅是最纯粹的。

民以食为天。他只有吃好喝好才有心情再去慢慢钓鱼。

...

...

“师叔，您尝一尝这个红油火锅。”

平阳子命人将火锅准备好后，请青莲道长吴全义坐在了上首的位置。

“嗯，看起来就很不错的样子。不过老夫已经很久没有吃过蜀中口味的火锅了，不知道味道能否适应。”

青莲道长吴全义这个时候已经是口水直流了。

不过他还要表现的矜持一些，得要注意一下自己的形象。

“您肯定可以适应的。毕竟您是地地道道的蜀中人。虽然您去到长安城十几年，但是这个味道肯定是忘不掉的。”

平阳子还是很了解青莲道长的。

在他看来，只要这个红油火锅吃起来，那就瞬间可以回忆起以前的过往。

“还有这些油碟，小料，可谓都是细节。您只管吃，我来伺候您。”

说罢，便开始不断给鸳鸯火锅里放入菜肴。

“哈哈哈，看起来确实很诱人。”

青莲道长吴全义吞了口吐沫，继而接道：“要是再有一杯冰奶茶就好了。”

“冰奶茶？”

平阳子显然听得一愣。

“额没什么，老夫是随口说说。”

说到这里，青莲道长吴全义又有些想念在终南山的日子了。

当时他们在终南山吃火锅烧烤的时候，旺财总会奉上各种口味的冰奶茶。

还别说，那种感觉是真的让人觉得很惬意。

“独在异乡为异客，每逢佳节倍思亲。”

青莲道长感慨道：“在长安的时候老夫想的就是这一口火锅，真的是很不容易啊，终于让老夫给等到了。”

...

...

不得不说蜀中火锅的口味还是和长安火锅截然不同的。

青莲道长吴全义只吃了一口酒感觉到了熟悉的感觉。

“回来了，全部都回来了。”

青莲道长吴全义此刻感受到了一种久违的感觉。

“啧啧啧，这红油火锅配上这油碟，味道简直是绝了。”

“师叔，你这一次回来应该不会再走了吧？”

如今平阳子的心态和一开始的时候已经完全不同了。

他很希望青莲道长吴全义能够一直待在青城山。

只要得到了青莲道长吴全义的指点，他的修为境界一定可以在短时间内突飞猛进。

不管是从什么角度来看，这都是稳赚不赔的。

“嗯，老夫这一次回来应该不会再走了。人上了年纪啊，就会想要落叶归根。”

青莲道长吴全义有滋有味的吃着火锅，很是怅然道：“虽说很多故人都已经逝去了，但还是有很多熟人在的。”

“等老夫歇两天之后还要去一次锦官城。锦官城里也有老夫的几个故人。”

“嗯，师叔名满天下，自然有很多人想要拜会师叔。有什么需要的地方请尽管跟我说，我一定会尽力配合好的。”

“放心好了，真有需要老夫会知会你的。不过眼下看来应该不会有什么太大的问题。”

青莲道长吴全义去到锦官城，最重要的一件事情就是给锦官城和青城山见设立一个传送法阵，并将锦官城和长安城间的传送法阵修缮好。如此一来，就能够随时和长安方面取得联系，以备不时之需。

...

...

青莲道长吴全义经历过与撒旦的大战，很清楚对抗黑暗势力要未雨绸缪。

虽然短时间内来看，他们并不会遭到太大的威胁，但还是应该尽可能的提前做好准备。

锦官城距离长安实在太远了，如果单纯依靠人力去传送信息实在是太慢了。

所以最好的选择是建立传送阵。

在离开长安城之前青莲道长就已经和山长沟通过了，只要他在锦官城把传送阵的架子搭起来，就可以迅速的和长安方面建立联系。

至于青城山和锦官城之间的传送阵，在青莲道长吴全义看来就不在话下了。

越是这种时候越是得要想的全面一些。

青莲道长吴全义既然要给道门弟子争取尽可能多的机会，就需要多付出一些。

虽说道门如今和儒家有不小的差距。但是青莲道长吴全义相信，随着他的努力，这种差距会不断的缩小。

哪里有什么岁月静好，无非有人在替你负重前行。

青莲道长很清楚他总归是会老的。所以他会希望尽可能帮助道门弟子提前进入到一种更强大的状态。

尤其是在青城山，在三清观！

青莲道长吴全义是从这里走出来的，自然要对三清观的道门弟子付出更多的心血。

但行好事，莫问前程。

在这个时刻青莲道长吴全义不求任何回报，只希望能够在人生暮年燃烧自己，照亮别人。

大周世界一定会越来越好的，道门也会越来越好的。

青莲道长吴全义对此深信不疑。
